SIGN LANGUAGE RECOGNITION

A Convolutional Neural Network (CNN) built to classify American Sign Language (ASL) alphabets from static hand gesture images, achieving high accuracy through architectural tuning and regularization techniques.

Technologies: Python, TensorFlow, Machine Learning, OpenCV, MediaPipe

  • Preprocessed and augmented the dataset with normalization, resizing, and transformations to improve model generalization and address class imbalance.
  • Evaluated performance using confusion matrix and classification report; used TensorBoard and visualizations to monitor and refine training progress.
  • System extended to support real-time sign language detection using webcam input for efficient hand tracking and gesture classification.

A cross-platform mobile app using Flutter and Supabase to manage secure, time-bound university elections with separate roles for admin and student users.
